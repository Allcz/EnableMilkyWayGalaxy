# EnableMilkyWayGalaxy

## Features
Enables uploading cluster generation to galaxy server even if you use mods or not your game save.
Limited this version(0.8.22) available

即使使用mod 或者 使用其他人的存档 也允许上传戴森球发电数据到银河系服务器。
限游戏版本 （0.8.22） 可用
